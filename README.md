# GradePath starter

A beginner-friendly GitHub Pages starter for a syllabus + grade calculator app.

## What this version includes
- Email/password auth with Firebase
- Profile fields: first name, last name, birthday
- Syllabus upload to Firebase Storage
- Manual category editing
- Target grade calculator
- Save class results to Firestore

## What to do next
1. Create a GitHub repo
2. Upload these files
3. Enable GitHub Pages
4. Create a Firebase project
5. Paste your Firebase config into `firebase-config.js`
6. Enable Auth, Firestore, and Storage
7. Add your Firestore and Storage rules
8. Test signup, login, upload, and calculate

## Later AI step
You will add a backend endpoint that takes the uploaded syllabus and returns extracted categories/weights.
